## Project Overview

**RetroSpace** is a complete nostalgic social media app with:
- 11 fully functional pages
- Skeuomorphic design (glossy, beveled, textured)
- 6 mood-based color themes
- MSN-style messenger
- Daily page diary system
- Rooms instead of followers
- Memory box archive
- Profile customization
